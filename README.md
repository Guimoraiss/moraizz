# Personal Portfolio / Link Hub

A modern pixel-art inspired personal website that works as my central hub on the internet. It brings together my social links, future portfolios, and projects in a clean glassmorphism interface with animated backgrounds and a light/dark theme.

## Features

- Pixel art animated backgrounds
- Light & Dark mode toggle
- Glassmorphism UI
- Responsive design
- Quick access to my social media
- Music player section
- Dedicated cards for my future portfolios
- Smooth animations and hover effects

## Built With

- HTML5
- CSS3
- JavaScript

## Purpose

This website is designed to be the starting point for everything I create online.

Instead of having a simple Linktree page, I wanted something that reflects my personality and interests while also serving as a gateway to my work as:

- Web Developer
- Video Editor
- Music Producer

As my projects grow, this website will continue evolving alongside them.
